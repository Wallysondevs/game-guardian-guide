# Game Guardian — Guia Completo 🎮

Guia completo de modificação de memória Android usando Game Guardian, em **Português Brasileiro**.

## 📖 Acesse online

**[wallysondevs.github.io/game-guardian-guide](https://wallysondevs.github.io/game-guardian-guide/)**

## 📚 Conteúdo

- O que é Game Guardian e como funciona
- Download e instalação (com e sem root)
- Permissões Root (Magisk e KernelSU)
- Interface e tipos de busca de memória
- Edição de memória e travamento de valores
- Scripts Lua — introdução e técnicas avançadas
- Ponteiros, dump de memória, automação
- Hacks populares com exemplos práticos
- Virtual Space / VMOS (sem root)
- Bypass Anti-Cheat
- Uso seguro e anti-ban
- Troubleshooting
- Ética e responsabilidade

## 🛠️ Tecnologias

- React 19 + Vite + TypeScript
- Tailwind CSS v4
- Framer Motion
- React Syntax Highlighter
- Wouter (hash routing)

## 🚀 Desenvolvimento local

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
